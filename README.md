# Student Profile Card App

A React application that displays reusable student profile cards using component composition and props.

## Features

- Reusable React components
- Student profile cards
- Dynamic badges
- Score progress bars
- Conditional rendering
- Props and destructuring
- Responsive card layout

## Technologies Used

- React
- Vite
- CSS3

## Project Structure

```txt
src/
├── App.jsx
├── App.css
├── components/
│   ├── Header.jsx
│   ├── StudentCard.jsx
│   ├── StudentList.jsx
│   ├── Badge.jsx
│   └── StatBar.jsx
```
